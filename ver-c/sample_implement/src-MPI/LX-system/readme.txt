LXでの実行の仕方



(1) mpiifort a_main.f90

    でコンパイルすると、実行ファイルa.outができる。


(2) スクリプトのサンプルを参考に、スクリプトを作成し、
   
  a.outがあるフォルダにコピーする。
  
  (実行するnode数、process数、実行時間、実行クラス(PL, PM, SLなど)
   
  は必要に応じて変更してください。
    
  なお、1つのクラスで実行できるプログラム数は1つまでです)



(3) jobの投入方法
    
  たとえば、Tokai_LX_PLというスクリプトに記載した内容でjobを投入する場合は、

    qsub Tokai_LX_PL
  とすると、jobが投入されます。



(4) jobの状況確認

    qstat

  と打つと、現在実行中のjobが確認できます。

  他の人の情報も見れますので、空いているクラスを確認てjobを投入する場合にも

  使ってください。



(5) jobの停止

  qstatで"自分"のjb IDを確認して、

  たとえば、5555ならば

    qdel 5555

  とすると、そのjobは停止されます。
(少し時間が掛かります)
